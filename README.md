# Tic Tac Toe
A simple terminal Tic Tac toe game
## How to play
start the executable in your terminal. It will imidietly start. It will ask you where you want to place your piece. You write this in a cordinate like system. X first Y second for example `0 1` will place it on the first line second row. First row will always be 0 and last row will always be 2. To do your move you just press enter. It will then print out how the board looks. 1 is player 1s pieces and 2 is player 2s pieces.
